== Description ==

Flag View is a lightweight WordPress plugin that lets you display beautiful country flags with interactive hover animations.

Key features:
- Display multiple country flags with names
- Three size options (small, medium, large)
- Fun hover animations (jump, shake, fade)
- Easy to use shortcode implementation
- Fully responsive design
- Lightweight and fast

Perfect for:
- Multilingual websites
- Travel blogs
- Country-specific content
- Educational websites
- Event pages with international attendees

== Installation ==

1. Upload the `flag-view` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the [flag_view] shortcode in your posts, pages, or widgets

== Frequently Asked Questions ==

= How do I use the plugin? =
Simply use the [flag_view] shortcode in any post, page, or widget. You can customize it with parameters like:
[flag_view countries="us,gb,fr" size="large" animation="jump"]

= Where can I get support? =
Please visit the [support forums](https://wordpress.org/support/plugin/flag-view/).

= Can I add more countries? =
Yes! Just add the appropriate flag PNG file to the flags directory and extend the country list in the code.
